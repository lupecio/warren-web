# Warren Test
Frontend using react to show the warren test features.

I had implantation four features:

* Statement
* Deposit
* Bill
* Withdraw

I think I could have done more resources:

* Best design and ux
* Pagination

## Installation

```sh
yarn
```
or

```sh
npm install
```

## Usage

```sh
yarn start
```
or

```sh
npm start
```
